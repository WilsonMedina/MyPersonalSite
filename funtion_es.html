¿Que es una función?

😀😀😀

Una función, académicamente hablando, es una relación establecida entre dos conjuntos (X) y (F(x)=Y) que asigna a cada valor del conjunto (X) (variable independiente) un único valor del segundo conjunto (F(x)=Y) (variable dependiente).

Pero también debemos comprender que es una relación, y la misma está dada por la correspondencia lógica entre los elementos de dos conjuntos A, B que forman parejas ordenadas, la formulación de una expresión que une dos o más objetos entre si establece una relación.

Podemos clasificar a las funciones por:

Funciones inyectivas: a cada elemento del conjunto de llegada (F(x)=Y) le corresponde como máximo un elemento del conjunto de partida (X).

Funciones sobreyectivas: a cada elemento del conjunto de llegada (F(x)=Y) le corresponde por lo menos un elemento del conjunto de partida (X).

Funciones biyectivas: son las funciones que cumplen las dos condiciones de las funciones inyectivas y sobreyectivas.

Todo lo anterior son las definiciones académicas como se lee en diferentes libros de variados autores, en este corto articulo tratare de explicar con un dialecto menos académico y un ejemplo practico como yo desgloso lo abstracto que puede ser para algunos estudiantes en su etapa inicial el tema de las funciones.

Diariamente aplicamos relaciones que terminan siendo funciones, porque hay que tener siempre muy conceptualizado que toda función es una relación, pero toda relación no es una función.

Cuando preparamos un jugo en casa antes de procesar el mismo relacionamos los ingredientes que se usan es aquí cuando estamos aplicando una relación, recordar que la relación esta dada por la correspondencia entre los elementos, no podemos relacionar para hacer un jugo, del conjunto de las frutas una fruta 🍓 con el conjunto de los líquidos pintura para pared no hay correspondencia lógica, pero si podemos relacionar la fruta 🍓 con agua, recordar que una función se usa para asignarle un valor único a una variable independiente (X) en nuestra preparación del jugo ese valor único seria nuestro jugo después de ser procesado, es esta máquina la que tiene la fórmula que se le aplicara a nuestros ingredientes, esa fórmula es simplemente girar una cuchillas a muy alta velocidad para que triture y haga homogéneo nuestra preparación, y es así como después de relacionar unos ingredientes los convertimos en una variable de partida (X) que pasará por una fórmula y nos terminará entregando una variable de llegada (F(x)=Y) que nos termina ejemplificando una función clasificada como inyectiva donde la variable dependiente o de llegada, nuestro jugo, le corresponde solo una variable independiente o de partida, nuestros ingredientes!
