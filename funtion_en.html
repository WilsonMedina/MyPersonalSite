What is a funtion?

😀😀😀

A function, academically speaking, is a relation established between two sets (X) and (F(x)=Y) that assigns to each value of the set (X) (independent variable) a unique value of the second set (F(x)=Y) (dependent variable).

But we must also understand that it is a relationship, and it is given by the logical correspondence between the elements of two sets A, B that form ordered pairs, the formulation of an expression that joins two or more objects together establishes a relationship.

We can classify functions by:

Injective functions: each element of the arrival set (F(x)=Y) corresponds to a maximum of one element of the starting set (X).

Surjective functions: each element of the arrival set (F(x)=Y) corresponds to at least one element of the starting set (X).

Bijective functions: they are the functions that fulfill the two conditions of the injective and surjective functions.

All of the above are the academic definitions as read in different books by various authors, in this short article I will try to explain with a less academic dialect and a practical example how I break down how abstract the subject can be for some students in their initial stage. of the functions.

Every day we apply relationships that end up being functions, because we must always be very conceptualized that every function is a relationship, but every relationship is not a function.

When we prepare a juice at home, before processing it, we relate the ingredients that are used, it is here when we are applying a relationship, remember that the relationship is given by the correspondence between the elements, we cannot relate to make a juice, from the set of fruits a fruit 🍓 with the set of liquids there is no logical correspondence, but if we can relate the fruit 🍓 with water, remember that a function is used to assign a unique value to an independent variable (X) in our juice preparation that Our juice would be of unique value after being processed. This formula is simply turning a blade at a very high speed so that it crushes and makes our preparation homogeneous, and that is how After relating some ingredients, we convert them into a starting variable (X) that will go through a formula and end up giving us a variable of arrival (F(x)=Y) that ends up exemplifying a function classified as injective where the dependent or arrival variable, our juice, only corresponds to an independent or starting variable, our ingredients!
